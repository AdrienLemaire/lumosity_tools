# lumosity_tools
Tools to get more efficient when training at Lumosity


# Installation

Build your virtual environment with python 3:

    $ mkvirtualenv -p /usr/bin/python3.4 lumosity

Install the dependencies:

    $ pip install -r requirements.txt
